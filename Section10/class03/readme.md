# NPM - Node Package Manager

* npm init
* npm install
    *--save-dev
    *--save-prod
    *-E: doesn't update the current version
    *^: updates minor and path
    *~: updates path only
    *@2.1.0: specify the version
    *@4.x | @4: last version of a major
* npm uninstall
* npm update
* npm ls
    *--depth=0
* npm outdated
